cURL Examples

Test the server from command line:

GET Request:

curl -v http://localhost:9000/

POST File Upload:

curl -X POST -F "file=@test.txt" http://localhost:9000/uploads

CGI Test:

curl http://localhost:9000/cgi-bin/test.py

DELETE Request:

curl -X DELETE http://localhost:9000/uploads/testfile.txt
